# 機能モジュールのプロジェクトの設定 {#top}

機能モジュールのプロジェクトで必要な設定を解説します。

機能モジュールのプロジェクトでの設定は、 application-core プロジェクトと infrastructure プロジェクトの設定を合わせたものとなります。

CSR 編の [application-core プロジェクトの設定](../../../csr/java/sub-project-settings/application-core-project-settings.md) および [infrastructure プロジェクトの設定](../../../csr/java/sub-project-settings/infrastructure-project-settings.md) を参考にしてください。
