# サブプロジェクトの個別設定 {#top}

サブプロジェクト単位で個別に設定が必要な内容について解説します。

ここで解説する設定はあくまで推奨設定であり、検討した上で変更することは問題ありません。
また、依存ライブラリについて、プロジェクトで必要なライブラリを適宜追加することも問題ありません。
