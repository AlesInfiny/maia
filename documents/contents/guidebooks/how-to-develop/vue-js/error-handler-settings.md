# エラーハンドラーの設定 {#top}

## グローバルエラーハンドラーの設定 {#global-error-handler}

業務フロー上発生が想定されない例外を捕捉し、ハンドリングするためのグローバルエラーハンドラーを設定します。

## カスタムエラーハンドラーの設定 {#custom-error-handler}

業務フロー上発生が想定されるエラーを補足し、ハンドリングするためのカスタムエラーハンドラーを設定します。
